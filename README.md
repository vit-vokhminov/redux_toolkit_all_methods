## Redux Toolkit

### Заготовки для использования

- `Стандартный reducers`
- `ExtraReducers`
- `Reselect`
- `RTK Query`

____

### Запуск

#### В первой консоли

```
npm i
npm start
```

#### Второй консоли

```
json-server --watch db.json --port 5000
```

#### При необходимости

```
pm i -g json-server
```
